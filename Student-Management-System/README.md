# Student Management System

## Description
A menu-driven Java project which focuses on a school system which manages 50 slots. It enables registration of a student, displaying complete student details, searching students by roll number and also finding the topper from the given set of students.

## Features
* Add a student
* Display all students
* Search students by roll numbers
* Find topper

## Technologies Used
* Java
* Object-Oriented Programming

## How to Run
1. Compile the program:
   javac StudentManagementSystem.java
2. Run the program:
   java StudentManagementSystem

## Concepts Used
* Classes and objects
* Array for storing student details
* Scanner (for input)
* Methods and constructors
