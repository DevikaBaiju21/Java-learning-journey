# Bank Management System

## Description
A menu-driven Java project which enables users to create an account, withdraw or deposit money and manage bank operations. The system supports up to 50 accounts.

## Features
* Create new accounts
* Deposit money
* Withdraw money
* Check for duplicate accounts
* Stores account details using an array

## How to Run
1. Compile the program:
   javac BankManagementSystem.java

2. Run the program:
   java BankManagementSystem

## Concepts Used
* Classes and objects
* Arrays
* Scanner (for input)
* Methods
* Searching in an array
